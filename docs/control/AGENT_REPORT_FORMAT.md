# Agent Report Format

Every agent work unit must produce a report.

```md
# Agent Report

## Task ID

## Durable Source Of Truth

## Task Scope

## Summary

## Files Inspected

## Files Changed

## Key Decisions

## Validation Run

## Validation Result

## Acceptance Criteria Status

## Storage Boundary Evidence

## Runtime Artifact Status

## Risks / Regressions

## Stop Condition Reached

## Open Questions

## Suggested Next Task
```

Reports must not rely on chat-only context.
